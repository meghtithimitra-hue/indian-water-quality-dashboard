# indian-water-quality-dashboard
R Shiny dashboard to visualize and explore Indian water quality data.

### Live App: https://meghtithimitra-mds202520.shinyapps.io/INDIAN_WATER_QUALITY/

### Summary

Interactive R Shiny dashboard to explore Indian water quality data (2021â€“2023) with spatial maps, parameter-wise analysis, correlation analysis, and a custom Water Quality Index (WQI). Built to make environmental data easy to explore for both technical and non-technical users.

### Key Features:

 State-wise map for parameter visualization (Temperature, pH, DO, BOD, etc.)

 Interactive charts by water body type and year

 Correlation analysis between any two parameters

 Custom WQI with weighted scoring and classification

 Clean data table with download option

### Tools:

R, Shiny, shinydashboard, plotly, leaflet, DT, dplyr, tidyr

### Run Locally
install.packages(c("shiny","shinydashboard","plotly","DT","dplyr","leaflet","tidyr"))
shiny::runApp()

### Why This Project

Demonstrates skills in interactive dashboards, data visualization, geospatial analysis, and turning data into a usable product.
